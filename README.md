# Power Output Analysis

Phân tích ảnh hưởng của các yếu tố môi trường và chủ quan lên công suất đầu ra của hệ thống điện mặt trời.

## Mục tiêu

Đánh giá ảnh hưởng của:

1. **Yếu tố môi trường (khách quan)**:

   - Bức xạ mặt trời (Irradiance)
   - Nhiệt độ môi trường (Temperature)
   - Độ ẩm (Humidity)

2. **Yếu tố chủ quan**:
   - Tình trạng tấm pin PV (thông qua điện trở cách điện, điện dung rò)
   - Hiệu suất bộ nghịch lưu DC-AC (thông qua số lần chuyển mạch, lỗi, cảnh báo)

Cac thong tin khac se bo sung sau.
